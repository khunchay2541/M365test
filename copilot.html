พบกับ Copilot
ผู้ช่วย AI สำหรับงานของคุณ



Microsoft 365 Copilot
เป็นฟีเจอร์ AI ที่ฝังอยู่ในแอปของ Microsoft 365 เช่น Word, Excel, PowerPoint, Outlook และ Teams เพื่อช่วยให้ผู้ใช้ทำงานได้เร็วขึ้นและมีประสิทธิภาพมากขึ้น
  • ช่วยงานในแอป Microsoft 365
  • เหมาะสำหรับผู้ใช้ทั่วไป
  • เน้นสร้างเอกสาร, สรุปข้อมูล
  • จุดเด่นคือใช้งานง่าย

สนใจติดต่อ


Copilot Studio
เป็นแพลตฟอร์มสำหรับนักพัฒนา และผู้ดูแลระบบในการสร้าง และปรับแต่ง Copilot ขององค์กรเอง โดยใช้เทคโนโลยีจาก Power Platform และ Azure OpenA
  • สร้าง Copilot แบบกำหนดเอง
  • เหมาะสำหรับนักพัฒนา / องค์กร
  • เน้นเชื่อมต่อข้อมูล, ปรับแต่ง AI
  • จุดเด่นคือปรับแต่งได้ละเอียดเด่นคือใช้งานง่าย

สนใจติดต่อ

แพ็กเกจที่เหมาะกับองค์กรของคุณ

Microsoft 365 Copilot Chat
Copilot Chat ให้บริการโดยไม่มีค่าใช้จ่ายเพิ่มเติมสำหรับผู้ใช้บัญชี Entra ทุกคนที่มีการสมัครใช้งาน Microsoft 365
  • ใช้การแชท AI ที่ขับเคลื่อนโดย GPT-4o ฟรีได้อย่างปลอดภัย
  • เข้าถึงและใช้เอเจนต์โดยคิดราคาตามการใช้งาน
  • รักษาการควบคุมด้าน IT รวมถึงการปกป้องข้อมูลองค์กรและการจัดการเอเจนต์
Microsoft 365 Copilot
เพิ่มผลิตภาพการทำงานของพนักงาน ทำให้กระบวนการทางธุรกิจง่ายขึ้น และเพิ่มมูลค่าที่วัดผลได้ทั่วทั้งองค์กรของคุณด้วยผู้ช่วย AI
  • ใช้ Copilot Chat ที่เชื่อมต่อกับข้อมูลงานของคุณและรับสิทธิ์การเข้าถึง AI การให้เหตุผลสำหรับการค้นคว้าและการวิเคราะห์ข้อมูล
  • เข้าถึง Microsoft 365 Copilot ในแอปต่างๆ เช่น Microsoft Teams, Outlook, Word, PowerPoint และ Excel
  • สร้างและใช้เอเจนต์ด้วย Copilot Studio
  • สัมผัสประสบการณ์การรักษาความปลอดภัย ความเป็นส่วนตัว และการปฏิบัติตามข้อบังคับระดับองค์กร
